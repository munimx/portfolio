# Munim Ahmad — Portfolio

Editorial-style portfolio built with Next.js, React, TypeScript, Tailwind, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

## Validate build

```bash
npm run lint
npm run build
```
